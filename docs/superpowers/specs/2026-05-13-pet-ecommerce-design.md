# 寵物電商網站設計文檔

## 概述
全方位寵物用品電商網站，涵蓋狗、貓、鳥、魚、小動物等所有寵物類型。前後端分離架構，溫馨可愛風視覺設計。

## 技術架構
- 前端：React + React Router + Zustand + Vite + 自訂 CSS
- 後端：Node.js + Express + Prisma ORM + PostgreSQL
- 認證：JWT + bcrypt
- 金流：Stripe
- 設計風格：溫馨可愛風（暖橘 #F5A623、粉紅 #FF6B6B、米白 #FFF3E0、柔和綠 #81C784）

## 資料庫設計
9 個核心資料表：users, categories, products, reviews, favorites, orders, order_items, coupons, payments

## 頁面結構
公開頁面（6頁）+ 會員頁面（4頁）+ 管理後台（4頁）

## 實作階段
7 個階段，預計 11 天完成
